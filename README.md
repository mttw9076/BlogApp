# BlogApp
Lekka i rozbudowywalna aplikacja blogowa stworzona w ASP.NET Core MVC z obsługą Identity, Entity Framework Core oraz wielojęzyczności (Localization).

Funkcje
✍️ Dodawanie, edycja i usuwanie postów

👤 Rejestracja i logowanie użytkowników (Identity)

🌍 Automatyczna zmiana języka na podstawie ustawień przeglądarki

📄 Obsługa widoków Razor

🗄️ Baza danych SQL Server (EF Core)

🎨 Responsywny interfejs oparty na Bootstrap

Technologie
.NET 8 / ASP.NET Core MVC

Entity Framework Core

Identity

Localization (Resources)

SQL Server

Bootstrap 5

Cel projektu
Stworzenie skalowalnej, nowoczesnej platformy blogowej z możliwością dalszej rozbudowy o komentarze, tagi, panel admina i API.
